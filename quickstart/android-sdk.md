## 简介

pre-dem-android 是由[七牛云](https://www.qiniu.com)发起和维护的针对 Android 的集用户体验监控及报障于一体的开源 SDK，具有无埋点集成，轻量级，高性能等优点

## 功能清单

| 功能 | 版本 |
| - | - |
| crash 监控 | v1.0.0 |
| HTTP 性能监控 | v1.0.0 |
| UI 卡顿监控 | v1.0.0 |
| 网络诊断 | v1.0.0 |
| 自定义事件上报 | v1.0.0 |
